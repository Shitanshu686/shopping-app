# Shopping App

A responsive shopping web application built using HTML, CSS, and JavaScript.

## Features
- Product Listing
- Add to Cart
- Increase/Decrease Quantity
- Remove Items
- Total Price Calculation
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Future Improvements
- Java Servlets
- JSP
- JDBC
- MySQL Database
- User Login
- Session Management
- Cookies
- Checkout System
